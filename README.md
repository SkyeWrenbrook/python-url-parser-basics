# Python URL Parser

This project shows how to extract parts of a URL using Python string slicing and indexing. 

## Why this matters

URL parsing is a foundational skill in cybersecurity. It is used in:

- Log analysis
- Threat detection
- Identifying suspicious domains

## Features

- Extract domain extension (.com, .org, etc.)
- Extract website name
- Handles common URL structures

## Example

Input:
https://exampleURL1.com

Output:

Extension: .com 

Website name: exampleURL1

## Skills Demonstrated
- Python
- String manipulation
- Problem-solving
- Basic parsing logic
